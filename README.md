# KBS 카이로스챗(KairosChat) 활용 교육 · 강의교안

KBS 임직원을 대상으로 한 **멀티 LLM 플랫폼 '카이로스챗'** 실무 활용 교육 강의교안입니다.
2026년 7월 10일, 총 2시간(120분) 과정 기준으로 제작되었습니다.

🔗 **배포 URL:** 아래 Vercel 링크 참고

## 커리큘럼 (2시간)

| 단계 | 시간 | 구분 | 주요 내용 |
| --- | --- | --- | --- |
| 도입 | 20분 | 오리엔테이션 | 멀티 LLM 개념, GPT·Claude·Gemini 비교, 활용 장점, 접속 안내 |
| 기본 | 40분 | 이론 | 카이로스챗 기본 기능, LLM별 특징·선택 기준, 크레딧(토큰), 데이터 보안 |
| 심화 | 40분 | 시연 & 실습 | 나만의 챗봇 만들기(스튜디오), 프롬프트 엔지니어링, RAG, 공유·배포 |
| 마무리 | 20분 | 정리 & Q&A | 요약, 현업 적용 미션, 질의응답 |

## 특징

- 🎬 **방송 실무 즉시 적용 프롬프트** — 복사 버튼으로 바로 사용 (취재 요약, 팩트 확인, 보도자료, 자막, 회의록 등)
- 🧠 **섹션별 인터랙티브 퀴즈** — 각 단계마다 1문항, 정답·해설 즉시 확인
- ⚡ **멀티 LLM 비교 카드** — 상황별 모델 선택 가이드
- 📱 반응형 · 다크 테마 랜딩페이지

## 기술 스택

- Next.js 15 (App Router)
- React 19
- Tailwind CSS v4
- Vercel 배포

## 로컬 실행

```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # 프로덕션 빌드
```
